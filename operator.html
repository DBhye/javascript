연산자

1.산술연산자(+, -, /, %)
- / : 나누어서 몫, % : 나누어서 나머지
2.증감연산자 (++,--)
- 증감연산자 : 변수의 앞 또는 뒤에 ++붙여 사용, 전위증가연산자, 후위증가연산자
- 감소연산자 : 변수의 앞 또는 뒤에 --붙여 사용, 전위감소연산자, 후위감소연산자
3. 비교(관계)연산자 : > , >= , < , <= , ==(equal), !=, - === , !== (자바스크립트에서 추가)
- 결과가 논리값 형태로 표현, true 또는 false를 출력
4. 논리연산자 : &&(논리 AND, 논리곱), ||(논리 OR, 논리합), ! (논리 NOT,논리부정)
- 논리연산자의 결과는 논리값(TRUE, FALSE 둘중의 하나)
- 비교연산자가 논리연산자보다 우선임 비교 > 논리
- 논리 AND가 논리 OR보다 우선임.
5. 대입연산자 : = 복합대입연산자(할당연산자,assignment : += -= *= /= %=)
